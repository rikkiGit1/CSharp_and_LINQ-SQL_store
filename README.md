# Store Sales Management Application
This C# Windows Forms application manages store sales by allowing users to log in, browse items, make purchases, track account balances, and view past transactions. 
It integrates a SQL Server database with LINQ-to-SQL and demonstrates how a desktop GUI interacts with backend logic and database operations.

## Features
User login and account management
Browse and purchase items
Track balances and update purchases in real time
View past transaction history
LINQ-to-SQL database integration
Windows Forms interface

## Program Flow
Users log in, select items to purchase, and the app updates the database and account balance in real time. 
Purchase history can be queried at any time, demonstrating how the GUI, backend, and data work together seamlessly.
